This page collects common student questions during setup, experiments, and submission.

## Which version should I use?

Use the version assigned by your instructor. If you were not told which one to use, start with [Choose a Version](../get-started/choose-version.md).

## What should I do first after launch?

Run one baseline experiment with the default settings unless your instructor gave different instructions.

## What if my results look different from someone else's?

That can happen. First confirm that you used the same settings and the same run length. Then compare what changed and what stayed the same before drawing conclusions.

## What if I cannot export or save my work?

Try again before closing the app. If it still fails, record exactly what you did, what you expected, and what happened, then tell your instructor. Use [Reports and Submission](../students/reports-and-submission.md) for the checklist.

## Where can I get another explanation of the task?

Use [Student First Steps](../students/first-steps.md) or the [student walkthroughs](../videos/student-walkthroughs.md).
