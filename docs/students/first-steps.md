---
title: First Steps
---

Use this page to complete your first successful Avida-ED run and avoid common
mistakes.

## Step 1: Launch

Your instructor will usually direct you to Avida-ED 4 unless your course still
depends on legacy materials such as an older
[lab manual](/curriculum/lab-manual) handout or a
[model lesson](/curriculum/model-lessons).

If you are unsure, start from the [Launch Avida-ED](/app) page.

## Step 2: Run one baseline experiment

1. Use default settings unless your assignment says otherwise.
2. Run the experiment long enough to show a clear pattern.
3. Write down:
   - what settings you used
   - what you observed
   - what question the run is supposed to help answer

## Step 3: Change one thing

Change exactly one assigned condition, then run again for the same amount of
time or number of steps. If you need a fuller assignment pattern, use
[Course Workflow](./course-workflow.md).

## Step 4: Compare and explain

A strong explanation answers:

- What changed between the runs?
- What evidence supports your conclusion?
- Why do you think that change produced the outcome you observed?

For examples of the kinds of questions instructors assign, review
[Quick Start](/curriculum/quick-start),
[Exercise 1](/curriculum/lab-manual/exercise-1), or the
[model lessons](/curriculum/model-lessons).

## Saving and exporting

If your assignment requires exported data or saved files, make sure the file
actually downloads before you close the app.

If exporting fails, use the [Troubleshooting](../support/troubleshooting.md)
page.

## If you are not sure what to do next

Move to [Course Workflow](./course-workflow.md) for the full assignment pattern,
or use [Videos](./videos.md) if you need a quick orientation before trying the
written steps again.
