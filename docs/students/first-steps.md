title: Student First Steps
---

Use this page to complete your first successful Avida-ED run and avoid common mistakes.

## Step 1: Launch

Your instructor will tell you whether to use Avida-ED 4 or Avida-ED 3.

If you are unsure, see [Choose a Version](../get-started/choose-version.md).

## Step 2: Run one baseline experiment

1. Use default settings unless instructed otherwise.
2. Start the run and let it proceed long enough to show a clear trend.
3. Write down:
   - what settings you used
   - what you observed

## Step 3: Change one thing

Change exactly one condition (as assigned), then run again for the same amount of time/steps.

## Step 4: Compare and explain

A strong explanation answers:

- What changed between the runs?
- What evidence supports your conclusion?
- Why do you think the change produced that outcome?

## Saving and exporting

If your assignment requires exported data or a saved file, follow your instructor’s file naming rules and confirm the file actually downloaded.

If you have trouble exporting, see the [Troubleshooting Checklist](../get-started/troubleshooting.md).
