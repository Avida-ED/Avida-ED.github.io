This page collects the instructor-facing video set referenced by the legacy site and meeting notes.

## Recommended structure for each video entry

- title
- one-sentence purpose
- estimated viewing time
- transcript link
- notes on when to use it in teaching

## Priority teacher-facing videos

- how to launch and verify classroom readiness
- how to run a first comparison activity
- how to structure student submissions
- how to troubleshoot common classroom problems

## Legacy resources to preserve

## Migration standard

When these videos are added here, each entry should also identify:

- which Avida-ED version it matches
- whether it is still current or legacy-only
- where the transcript lives

Until the video set is finalized, pair any posted video with a transcript page under `videos/transcripts/`.
