---
title: System Requirements
---

Avida-ED runs in a modern web browser. For classroom use, reliability and consistency matter more than raw performance.

## Supported devices

- Desktop and laptop computers are recommended for activities that include data export and reporting.
- Tablets may work for guided activities, but can be less convenient for file handling and multi-window workflows.

## Browser guidance

Use a current version of:

- Chrome (recommended)
- Firefox
- Safari (recent versions)
- Edge

Avoid running Avida-ED in embedded browsers inside other apps.

## Classroom planning checklist

Before a class session:

1. Verify the school network allows access to the Avida-ED launch URL.
2. Confirm students can save/export files if your activity requires it.
3. Test one complete run of your intended activity on the same device type students will use.
4. Keep the [Troubleshooting Checklist](./troubleshooting.md) available.

## Accessibility note

If students use assistive technologies (screen readers, keyboard-only navigation), test the workflow in advance and report any barriers via the Support page.
