---
title: First Experiment
---

This guided activity is designed to confirm that Avida-ED launches correctly and to demonstrate the basic workflow: **run → observe → compare → explain**.

## Goal

Run two short experiments with different conditions and compare outcomes.

## Step 1: Launch and orient

1. Launch Avida-ED (v4 recommended).
2. Identify where you can:
   - start/stop a run
   - view population status or summary metrics
   - change experimental conditions
   - reset or create a new run

If you cannot find these controls, stop and use the [Troubleshooting Checklist](./troubleshooting.md).

## Step 2: Run a baseline experiment

1. Use default settings.
2. Run long enough to see stable trends (your teacher may specify a time/step count).
3. Record:
   - the setting name(s) you used
   - a short description of what you observed

## Step 3: Change one condition

Choose one change your instructor allows (examples):

- increase mutation rate
- change the environment/rewards
- modify a constraint that affects reproduction or survival

Run again and record the same observations.

## Step 4: Compare and explain

In 3–6 sentences:

- What changed between the runs?
- What stayed the same?
- What is your best explanation for the difference?
