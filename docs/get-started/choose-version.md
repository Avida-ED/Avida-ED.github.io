---
title: Choose a Version
---

Avida-ED is available in two web-based versions. Most classes should start with **Avida-ED 4** unless you have a specific reason to use legacy materials.

## Recommended default: Avida-ED 4

Use Avida-ED 4 if you want:

- the current supported classroom experience
- the best compatibility with modern browsers and devices
- new or updated instructional materials as they are released

**Launch:** use the homepage button for Avida-ED 4 or go directly to `/avida-ed4/` on GitHub Pages.

## Legacy option: Avida-ED 3

Use Avida-ED 3 if you need:

- compatibility with existing lesson plans or worksheets built around v3
- continuity with previously taught materials in your school/district

**Launch:** use the homepage button for Avida-ED 3 or go directly to `/avida-ed3/` on GitHub Pages.

## If you are unsure

Start with Avida-ED 4, run the [First Experiment](./first-experiment.md), and only switch to v3 if a specific lesson requires it.
