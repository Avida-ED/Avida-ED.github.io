Avida-ED helps students learn evolution and the nature of science by running **digital evolution experiments** and analyzing outcomes with data.

## Fast path (10 minutes)

1. **Choose a version:** start with [Choose a Version](./choose-version.md).
2. **Check your device/browser:** see [System Requirements](./system-requirements.md).
3. **Run your first experiment:** follow [First Experiment](./first-experiment.md).
4. If something fails, use the [Troubleshooting Checklist](./troubleshooting.md).

## Who is this for?

- **Teachers:** start with [Teacher Quick Start](../teachers/quick-start.md).
- **Students:** start with [Student First Steps](../students/first-steps.md).

## What you will be able to do

With Avida-ED, you can:

- set up experimental conditions (e.g., mutation rates, environment/rewards)
- observe population change over time
- compare experimental runs
- export results and data for analysis and reporting
