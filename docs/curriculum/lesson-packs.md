This page is the landing area for structured lesson packs migrated from the legacy site.

## Priority lesson-pack structure

Each lesson pack should include:

- one-sentence teaching goal
- recommended version of Avida-ED
- estimated class time
- student-facing instructions
- teacher notes and common pitfalls
- expected deliverables

## Initial pack types to migrate first

### Intro to natural selection

- Audience: first-time Avida-ED users
- Focus: baseline run plus one comparison
- Deliverable: short explanation using observations from the run

### Mutation and fitness

- Audience: classes comparing conditions across groups
- Focus: changing mutation-related settings and interpreting consequences
- Deliverable: comparison table or graph plus explanation

### Experimental design with digital evolution

- Audience: longer inquiry-based units
- Focus: controlled comparisons, claims, evidence, and reasoning
- Deliverable: lab-style report or presentation
