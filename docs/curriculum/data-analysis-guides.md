---
title: Data Analysis Guides
---

Students learn more from Avida-ED when they move beyond "what happened" to "what evidence supports the claim."

## Core analysis questions

Ask students to answer:

- What changed between runs?
- What stayed the same?
- What pattern did you observe?
- What evidence supports your conclusion?
- What alternative explanation should be considered?

## Useful outputs

- a comparison table across runs
- a simple graph from exported data
- screenshots of key views with annotations
- a short CER response: claim, evidence, reasoning

## Common analysis errors

- comparing runs of different lengths without accounting for that difference
- changing multiple variables at once
- describing trends without citing data
- treating a single run as decisive proof
