This page converts the legacy Avida-ED lab book area into an HTML-first index with stable links to archived files.

## Recommended use

Treat these files as **legacy reference materials** that still contain high-value activity structure, especially for instructors adapting older courses.

Use Avida-ED 4 as the default platform unless a specific activity still requires a legacy workflow.
