This page is an HTML-first migration of the first lab book exercise and the earlier student handout on mutation-driven genetic variation.

## Student learning goals

Students should be able to:

- explain what it means to say that mutations occur at random
- explain how mutations during reproduction generate genotypic variation in a population
- distinguish between the outcome of one run and the distribution seen across many runs

## Guiding questions

- How does genetic sequence variation originate in a population?
- Is there a pattern to how mutations occur?
- Could we predict which mutation an offspring will have?

## Background

Avidians, like biological organisms, can show both genotypic and phenotypic variation. In this exercise, the focus is on **genotypic variation** produced by mutation during reproduction.

The legacy materials emphasized a few key ideas:

- mutation is a fundamental source of variation
- in Avida-ED, mutation is modeled as substitution mutation
- if mutation did not occur, offspring would inherit the exact same genome as the parent
- different mutation events across many reproductions create population-level variation

For instructional simplicity, the legacy lab book notes that Avida-ED uses substitution mutations only, not the full range of biological mutation types.

## Suggested workflow

### Part 1: Make predictions

Before students run the exercise, ask them to predict:

- whether the same mutation will happen every time
- whether the same number of mutations will happen every time
- how mutation position and identity might be distributed across many runs

This prediction step is important because the later analysis depends on comparing expectations against observed distributions.

### Part 2: Run one pilot observation

Use the Organism view to observe reproduction with a mutation rate set for the exercise.

Students should:

1. load the ancestor organism
2. run one reproduction
3. record where mutations occurred, if any
4. compare their result to a classmate's result

## Questions after the pilot

- Did the same mutations occur for both students?
- Did the same number of mutations occur?
- What would you expect if the exercise were repeated several more times?

### Part 3: Repeat across several runs

Have students repeat the reproduction process for several independent replicates using the same ancestor and the same mutation-rate setting.

Students should record:

- genome positions where differences occurred
- the final mutated instruction state
- the total number of mutations per run

### Part 4: Pool the class data

The lab book treated the class as a larger data set so that students could distinguish:

- individual-run variability
- the broader distribution estimated from many runs

That distinction is one of the most valuable scientific reasoning outcomes in the activity.

## Discussion prompts

Use prompts like these from the legacy materials:

- How does the setup demonstrate that mutations are random?
- Did every student observe the same mutations and the same number of mutations?
- How would you describe the distribution of mutation positions, mutation identities, and mutation counts?
- How would the results differ if the mutation rate were lower, such as 5% instead of 10%?
- In what ways is mutation in Avida-ED similar to biological mutation?
- In what ways is it different?
- What are the limits of this model?

## Teaching notes

This lesson works well because it forces students to separate two ideas that are often confused:

- a random event at the level of individual mutations
- a predictable distribution that emerges across many observations

That makes the exercise a strong starting point for later work on variation, selection, and population change.
