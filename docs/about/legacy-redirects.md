This page records the intended destination for the highest-value legacy MSU URLs.

## Core redirects

- `/index.html` -> `/`
- `/curriculum/` -> `/docs/curriculum/`
- `/support/` -> `/docs/support/`
- `/instructor-faqs/` -> `/docs/support/instructor-faq/`
- `/student-faqs/` -> `/docs/support/student-faq/`
- `/digital-evolution/` -> `/docs/about/research-background/`
- `/about-us/` -> `/docs/about/`
- `/download-avida-ed/` -> `/docs/get-started/offline-and-download/`
- `/app/` -> `https://avida-ed.github.io/avida-ed3/`
- `/app4/` -> `https://avida-ed.github.io/avida-ed4/`

## Archive or deferred decisions

- `/version-history/` -> archive or future release-notes page
- `/vintage-archive/` -> archive-only destination
- `/news/` -> archive or history page if retained
- workshop/event pages -> archive if historically valuable; otherwise retire

## Implementation note

If the MSU domain can do server-side redirects, prefer that. If not, provide a minimal transition page for any high-traffic legacy URLs that cannot be redirected automatically.
