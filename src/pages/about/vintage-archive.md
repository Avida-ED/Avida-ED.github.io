### Changelog symbol key (legacy)

- `check` = bug fix
- `change` = change/update
- `plus` = feature addition
- `minus` = feature removal

#### 3.0.0 (web) released 2017-06-16

Thanks to the beta testers noted on the legacy page.

##### Key changes between Avida-ED 2.0 and 3.0

- **Now on the Web**: Avida-ED runs cross-platform in the browser.
- **Embedded media**: integrated tutorial/help video support in the web
  interface.
- **Export/Save behavior updates** for browser-based workflows.
- **Freezer items and workspaces** changed to zip-based packaging.

#### 3.0 BETA (web) released 2016-06-18

A beta of the next major version was released at Evolution 2016. The legacy
page described this as browser-based and cross-platform.

### 2.x releases

#### 2.0.41 (Mac) released 2014-01

- Golden release version.
- Fixed bug causing demo-mode populations to fail exact replay.

#### 2.0 BETA (Mac) released 2013-02

Major under-the-hood revision for performance and future extensibility, plus
several user-facing improvements.

### 1.x and earlier

#### 1.2.0 (Mac and PC) released 2011-09

- Updated OS X support for 10.6 (Snow Leopard) and 10.7 (Lion).
- Updated PC support for Windows 7.

#### Multidish BETA (Mac) released 2010 spring

- Experimental multidish version allowing stitched dishes with different
  environments.

#### 1.0.1 (Mac) released 2007-11-02

- Updated OS X build to run natively on Intel Macs.

#### 1.0 (Mac and PC) released 2007-06-19

Avida-ED public distribution release announced at the Society for the Study of
Evolution meeting in Christchurch, New Zealand.
